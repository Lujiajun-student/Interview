# 项目功能

1. 数据统计页面。统计课程总数、学生总数、学习时长、活跃学生数。课程分类饼图，使用ECharts展示各分类的课程占比；最新课程列表，展示新添加的课程。
2. 课程页面。分页展示课程信息，支持关键词搜索、分类筛选和状态筛选；CRUD操作，对课程进行增删改查。状态切换。
3. 学生页面。支持分页、关键词搜索、班级筛选。对学生也有增删改查操作，学生可以关联多个课程。
4. 课程总结页面。加载后端的md文件数据，根据标题层级自动生成目录，并通过知识图谱来可视化展示知识点关系。
5. AI咨询页面。多回话管理，能够创建、切换和关闭多个聊天会话。
