# 1. 绝对路径用什么符号表示，当前目录、上层目录用什么表示，主目录用什么表示，切换目录用什么命令

绝对路径：`/`。如果以`/`开头，就会从根目录开始搜索。

当前目录：`./`。

上层目录：`../`。

主目录：`~`。登录用户的个人文件夹。

切换目录：`cd`。

返回上一级：`cd ..`。

# 2. 怎么查看当前进程，怎么执行退出，怎么查看当前路径

查看当前进程用ps或者top，ps是查看当前一瞬间的进程状态，top是持续监控。

退出进程使用Ctrl+C。杀死进程先ps查找进程，然后kill PID就行。

查看路径用pwd。

# 3. 怎么清屏，怎么退出当前命令，怎么执行睡眠，怎么查看当前用户id，怎么查看帮助

清屏使用clear。退出当前命令使用Ctrl+C。睡眠使用sleep。查看用户id用id命令。查看帮助使用--help或者man。

# ls命令执行什么功能，有什么参数

ls用来列出当前目录的文件和文件夹。

| 参数 | 全称           | 描述                         |
| ---- | -------------- | ---------------------------- |
| -a   | all            | 显示所有文件，包含隐藏文件。 |
| -l   | long           | 查看文件和文件夹的详细信息。 |
| -h   | human-readable | 易读格式。                   |
| -t   | time           | 按时间排序。                 |
| -r   | reverse        | 反向排序，搭配其他排序方法。 |
| -S   | size           | 按文件格式排序。             |
|      |                |                              |

# 5. 查看文件的命令

使用`cat file.txt`就能查看文件。

或者用`nl file.txt`能够额外显示行号。

# 6. 常见的Linux命令

mkdir：创建文件夹。

rm：删除文件。

cp：复制文件或目录。

mv：移动文件。

touch：创建新文件。

find：搜索文件。

ip addr：查看ip地址。

curl：命令行下载工具。

ping：测试网络连通性。
